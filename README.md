# 📸 Automix ProductImage Pro

**Automix ProductImage Pro** هو نظام ذكي ومتكامل (SaaS) مصمم خصيصاً لمساعدة تجار التجزئة والمتاجر الإلكترونية في المملكة العربية السعودية (منصات سلة، زد، ونون) على معالجة، تحسين، وصناعة صور المنتجات باحترافية استوديو عالية وخلال ثوانٍ معدودة، لضمان مطابقتها لأعلى المعايير البصرية المحلية.

---

## ✨ المميزات الرئيسية (Key Features)

- **🎨 محرك تحسين الإضاءة الفائق (CLAHE Engine):** يعالج الظلال الذاتية، يزيل الغمامة والضباب البصري، ويبرز ألوان وتفاصيل المنتج الحقيقية والمشجعة على الشراء فوراً.
- **📐 تحجيم ذكي مرن حسب نوع المنصة:** توفير خيارات تحجيم مسبقة الأبعاد تناسب متطلبات المتاجر السعودية:
  - المربع القياسي ($1024 \times 1024$ بكسل) - لمنصات سلة، زد، ونون.
  - العمودي الإعلاني ($1350 \times 1080$ بكسل) - لقصص المتاجر وأقسام الملابس والأزياء.
  - الأفقي العريض ($1200 \times 628$ بكسل) - للبنرات والواجهات الإعلانية.
- **⬜ خلفية بيضاء نقية 100%:** يضمن توليد لوحة خلفية بلون أبيض ناصع تماماً ($255, 255, 255$) بدون أي شوائب لتبرز المنتجات بشكل احترافي.
- **✂️ اقتصاص تلقائي محمي (Auto-Crop):** خوارزمية ذكية لعزل الحواف والأطراف الفارغة وتركيز الكاميرا على المنتج، مع نظام حماية مخصص للهواتف المحمولة لتجنب القص الجائر والزوم الخاطئ.
- **🔄 استعراض الفحص (Before / After):** واجهة مقارنة بصرية حية تعرض الصورة الأصلية بجانب الصورة المعالجة قبل التحميل لتسهيل الفحص والتدقيق الفني.
- **📦 خيارات تحميل مرنة:** دعم التحميل العادي الفردي لكل صورة على حدة، أو التحميل الجماعي الدفعي لجميع الصور معاً في ملف مضغوط واحد (`.zip`) لتوفير الوقت وبيانات الجوال.
- **🔑 نظام تفعيل أمني ومؤتمت:** إدارة صلاحيات واستهلاك باقات المشتركين عبر رخص تفعيل فريدة يتم توزيعها تلقائياً بالربط مع منصات البيع الرقمي (Payhip / Gumroad).

## 📂 الهيكل البرمجي للمشروع (Project Structure)

تم تصميم المشروع بهيكلية برمجية منظمة، نظيفة، وقابلة للتوسع المستقبلي (Scalable Architecture):

```text
ProductImage-Pro/
│
├── core/
│   ├── __init__.py
│   ├── image_engine.py     # محرك معالجة وتحسين الصور (OpenCV & NumPy)
│   └── auth_manager.py     # نظام التحقق من الأكواد وإدارة باقات المشتركين
│
├── app.py                  # الملف الرئيسي لبناء وتطوير واجهة المستخدم (Streamlit UI)
├── requirements.txt        # الحزم والمكتبات البرمجية المطلوبة للتشغيل
└── README.md               # التوثيق البرمجي والتسويقي للمشروع
```
